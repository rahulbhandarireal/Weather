ClimaTrip – AI Travel & Weather Planner






ClimaTrip is an AI-powered travel and weather planner that generates personalized travel itineraries and provides real-time weather updates for destinations around the world. It helps travelers plan smarter, safer, and more enjoyable trips.

Features

Personalized Travel Plans: AI-generated itineraries based on user preferences.

Real-Time Weather Forecast: Accurate weather updates for your destination.

Secure Authentication: Spring Security ensures safe login and signup.

Intelligent Recommendations: Integration with OpenAI API for travel tips.

Responsive UI: Works across devices and screen sizes.

Technology Stack

Backend: Spring Boot (Java)

Frontend: JavaScript / HTML / CSS

APIs: OpenAI API, Weather API

Deployment: Vercel

Getting Started
Prerequisites

Java 17+

Maven

Node.js & npm (if building frontend locally)

Installation
# Clone the repository
git clone https://github.com/rahulbhandarireal/Weather.git
cd Weather

# Build backend
mvn clean install

# Run backend server
mvn spring-boot:run


Access the live frontend: https://clima-trip.vercel.app/

Usage

Sign up or log in securely.

Enter your travel destination and preferences.

Generate a personalized travel plan.

View real-time weather for your destination.

Save or export your trip plan.
